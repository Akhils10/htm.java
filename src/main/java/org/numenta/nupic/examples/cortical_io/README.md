# Example Demos
_(Illustrating the integration of [NuPIC](https://github.com/numenta/nupic) & [Cortical.io](http://cortical.io))_

This directory is the collecting place for programs which demonstrate the use of Cortical.io's Java API together with NuPIC.

1. **Fox Eats Demo** - Java version of Subutai Ahmad's 2013 Fall Hackathon project which integrates Cortical.io's API to determine semantic similarity between the diet of animals. ([Executable Jar](http://cognitionmission.com/FoxEatsDemo.jar))

2. **Breaking News Demo** - Java version of the Cortical.io team's 2015 Spring Hackathon project demonstrating Trend Analysis and Tracking of tweets from Twitter. ([Executable Jar](http://cognitionmission.com/breaking-news-demo-1.0.0.jar))
